# v1.0.0 - RetailNext Outfit Assistant

Release date: 2026-02-07

## Highlights

- Delivered end-to-end storefront demo with FastAPI backend and SQLite persistence.
- Added natural-language query search with embedding-based retrieval.
- Added image upload matching flow with AI-assisted query generation.
- Added voice-to-text support for search input.
- Implemented `Check Your Match` scoring with detailed, per-signal explanations.
- Added modal-based expanded judgement view for better readability.
- Hardened local runtime reliability:
  - default dev port updated to `8001`
  - automatic free-port selection
  - same-origin API fetch behavior to avoid mixed-port failures
- Added complete docs pack (architecture, scoring explanation, troubleshooting).

## Included Features

- Home feed browsing and gender filtering
- Personalized recommendation sessions
- Match-check persistence in SQLite
- Health endpoint and image serving endpoint
- OpenAI timeout guards and graceful fallback behavior

## Key Technical Notes

- API server: `app/api_server.py`

- Frontend scripts:
